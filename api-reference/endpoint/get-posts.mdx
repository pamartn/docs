---
title: "Get Posts"
api: "GET /api/v1/posts"
description: "Fetch posts from a LinkedIn profile with engagement metrics"
---

## Endpoint

```
GET /api/v1/posts
```

## Authentication

Include your API key in the request header:

```bash
X-API-Key: your-api-key-here
```

## Parameters

<ParamField query="profileUrlOrUrn" type="string" required>
  LinkedIn profile URL or URN.
  
  Examples:
  - `https://www.linkedin.com/in/username`
  - `urn:li:member:123456789`
</ParamField>

<ParamField query="count" type="integer" default="10">
  Number of posts to fetch.
  
  - Minimum: 1
  - Maximum: 50
  - Default: 10
</ParamField>

## Response

<ResponseField name="posts" type="array">
  Array of LinkedIn posts

<Expandable title="Post object">
  <ResponseField name="urn" type="string">
    Unique post identifier
  </ResponseField>
  
  <ResponseField name="text" type="string">
    Post content/text
  </ResponseField>
  
  <ResponseField name="publishedAt" type="string">
    ISO 8601 timestamp of when the post was published
  </ResponseField>
  
  <ResponseField name="numberOfLikes" type="integer">
    Total likes count
  </ResponseField>
  
  <ResponseField name="numberOfComments" type="integer">
    Total comments count
  </ResponseField>
  
  <ResponseField name="numberOfReposts" type="integer">
    Total reposts/shares count
  </ResponseField>
  
  <ResponseField name="imageUrl" type="string">
    URL of post image (if present)
  </ResponseField>
  
  <ResponseField name="videoUrl" type="string">
    URL of post video (if present)
  </ResponseField>
</Expandable>
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET "https://api.fetchin.io/api/v1/posts?profileUrlOrUrn=https://www.linkedin.com/in/williamhgates&count=5" \
  -H "X-API-Key: your-api-key-here"
```

```javascript JavaScript
const response = await fetch(
  'https://api.fetchin.io/api/v1/posts?profileUrlOrUrn=https://www.linkedin.com/in/williamhgates&count=5',
  {
    headers: {
      'X-API-Key': 'your-api-key-here'
    }
  }
);

const posts = await response.json();
console.log(posts);
```

```python Python
import requests

headers = {
    'X-API-Key': 'your-api-key-here'
}

params = {
    'profileUrlOrUrn': 'https://www.linkedin.com/in/williamhgates',
    'count': 5
}

response = requests.get(
    'https://api.fetchin.io/api/v1/posts',
    headers=headers,
    params=params
)

posts = response.json()
print(posts)
```

```php PHP
<?php

$apiKey = 'your-api-key-here';
$profileUrl = 'https://www.linkedin.com/in/williamhgates';
$count = 5;

$url = "https://api.fetchin.io/api/v1/posts?profileUrlOrUrn=" . urlencode($profileUrl) . "&count=" . $count;

$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, $url);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'X-API-Key: ' . $apiKey
]);

$response = curl_exec($ch);
$posts = json_decode($response, true);

curl_close($ch);
print_r($posts);
?>
```

</CodeGroup>

## Notes

<Info>
  This endpoint counts as 1 request toward your monthly quota, regardless of the `count` parameter value.
</Info>

<Warning>
  Posts are returned in reverse chronological order (newest first).
</Warning>

<Tip>
  Cache results when possible to avoid redundant requests for the same profile.
</Tip>
