## Endpoint

```
GET /api/v1/profile
```

## Authentication

Include your API key in the request header:

```bash
X-API-Key: your-api-key-here
```

## Parameters

<ParamField query="profileUrlOrUrn" type="string" required>
  LinkedIn profile URL or URN.
  
  Examples:
  - `https://www.linkedin.com/in/username`
  - `urn:li:member:123456789`
</ParamField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET "https://api.fetchin.io/api/v1/profile?profileUrlOrUrn=https://www.linkedin.com/in/williamhgates" \
  -H "X-API-Key: your-api-key-here"
```

```javascript JavaScript
const response = await fetch(
  'https://api.fetchin.io/api/v1/profile?profileUrlOrUrn=https://www.linkedin.com/in/williamhgates',
  {
    headers: {
      'X-API-Key': 'your-api-key-here'
    }
  }
);

const profile = await response.json();
console.log(profile);
```

```python Python
import requests

headers = {
    'X-API-Key': 'your-api-key-here'
}

params = {
    'profileUrlOrUrn': 'https://www.linkedin.com/in/williamhgates'
}

response = requests.get(
    'https://api.fetchin.io/api/v1/profile',
    headers=headers,
    params=params
)

profile = response.json()
print(profile)
```

```php PHP
<?php

$apiKey = 'your-api-key-here';
$profileUrl = 'https://www.linkedin.com/in/williamhgates';

$url = "https://api.fetchin.io/api/v1/profile?profileUrlOrUrn=" . urlencode($profileUrl);

$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, $url);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'X-API-Key: ' . $apiKey
]);

$response = curl_exec($ch);
$profile = json_decode($response, true);

curl_close($ch);
print_r($profile);
?>
```

</CodeGroup>

## Notes

  This endpoint counts as 1 request toward your monthly quota. Failed requests (errors) do not count against your quota.
</Info>

</Warning>

<Tip>
