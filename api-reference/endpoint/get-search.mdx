## Endpoint

```
GET /api/v1/search
```

## Authentication

Include your API key in the request header:

```bash
X-API-Key: your-api-key-here
```

## Parameters

<ParamField query="keyword" type="string" required>
  Search keyword to find LinkedIn profiles.
  
  Examples:
  - `software engineer`
  - `marketing manager`
  - `John Doe`
  - `Google`
</ParamField>

## Response

<ResponseField name="results" type="array">
  Array of matching LinkedIn profiles

<Expandable title="Search result object">
  <ResponseField name="name" type="string" required>
    Full name of the LinkedIn member
  </ResponseField>
  
  <ResponseField name="profileUrl" type="string" required>
    LinkedIn profile URL
  </ResponseField>
  
  <ResponseField name="profilePictureUrl" type="string">
    Profile picture URL
  </ResponseField>
  
  <ResponseField name="headline" type="string">
    Professional headline
  </ResponseField>
  
  <ResponseField name="location" type="string">
    Geographic location
  </ResponseField>
</Expandable>
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET "https://api.fetchin.io/api/v1/search?keyword=software%20engineer" \
  -H "X-API-Key: your-api-key-here"
```

```javascript JavaScript
const response = await fetch(
  'https://api.fetchin.io/api/v1/search?keyword=software%20engineer',
  {
    headers: {
      'X-API-Key': 'your-api-key-here'
    }
  }
);

const results = await response.json();
console.log(results);
```

```python Python
import requests

headers = {
    'X-API-Key': 'your-api-key-here'
}

params = {
    'keyword': 'software engineer'
}

response = requests.get(
    'https://api.fetchin.io/api/v1/search',
    headers=headers,
    params=params
)

results = response.json()
print(results)
```

```php PHP
<?php

$apiKey = 'your-api-key-here';
$keyword = 'software engineer';

$url = "https://api.fetchin.io/api/v1/search?keyword=" . urlencode($keyword);

$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, $url);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'X-API-Key: ' . $apiKey
]);

$response = curl_exec($ch);
$results = json_decode($response, true);

curl_close($ch);
print_r($results);
?>
```

</CodeGroup>
