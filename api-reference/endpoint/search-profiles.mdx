## Endpoint

```
GET /api/v1/profiles/search
```

## Authentication

Include your API key in the request header:

```bash
X-API-Key: your-api-key-here
```

## Parameters

<ParamField query="limit" type="number" required>
  Number of profiles to return (1-100). Determines the credits charged: each result costs 100 credits.

  Example: `limit=10` costs 1,000 credits.
</ParamField>

<ParamField query="keyword" type="string">
  Search across headlines, job descriptions, and skills.

  Examples:
  - `machine learning engineer`
  - `product manager`
  - `data scientist`
</ParamField>

<ParamField query="sortBy" type="string" default="relevance">
  Sort order for results.

  - `relevance` — Best match first, boosted by follower count
  - `followers` — Most followed profiles first
</ParamField>

<ParamField query="location" type="string">
  Filter by geographic location.

  Examples:
  - `San Francisco`
  - `France`
  - `London, United Kingdom`
</ParamField>

<ParamField query="jobTitle" type="string">
  Filter by current job title.

  Examples:
  - `CTO`
  - `Product Manager`
  - `Software Engineer`
</ParamField>

<ParamField query="companyName" type="string">
  Filter by current company.

  Examples:
  - `Google`
  - `Stripe`
  - `Goldman Sachs`
</ParamField>

<ParamField query="skills" type="string">
  Comma-separated list of skills to filter by. Profiles matching at least one of the listed skills will be returned.

  Example: `Python,React,AWS`
</ParamField>

<ParamField query="minFollowers" type="number">
  Minimum follower count. Use with `maxFollowers` for a range.

  Example: `1000`
</ParamField>

<ParamField query="maxFollowers" type="number">
  Maximum follower count. Use with `minFollowers` for a range.

  Example: `50000`
</ParamField>

<ParamField query="creator" type="string">
  Filter by LinkedIn Creator mode status. Pass `"true"` or `"false"`.
</ParamField>

<ParamField query="influencer" type="string">
  Filter by LinkedIn Influencer status. Pass `"true"` or `"false"`.
</ParamField>

<ParamField query="premium" type="string">
  Filter by LinkedIn Premium subscription. Pass `"true"` or `"false"`.
</ParamField>

<ParamField query="countryCode" type="string">
  ISO country code to filter by geography.

  Examples: `US`, `FR`, `DE`, `IN`
</ParamField>

<ParamField query="excludeUrns" type="string">
  Comma-separated LinkedIn URNs to exclude from results. Useful for deduplication when making multiple calls.

  Example: `urn:li:member:123,urn:li:member:456`
</ParamField>

<Warning>
  At least one search filter is required (`keyword`, `location`, `jobTitle`, `companyName`, `skills`, `creator`, `influencer`, `premium`, or `countryCode`).
</Warning>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET "https://api.fetchin.io/api/v1/profiles/search?keyword=machine%20learning&countryCode=US&minFollowers=500&limit=5" \
  -H "X-API-Key: your-api-key-here"
```

```javascript JavaScript
const params = new URLSearchParams({
  keyword: 'machine learning',
  countryCode: 'US',
  minFollowers: '500',
  limit: '5',
});

const response = await fetch(
  `https://api.fetchin.io/api/v1/profiles/search?${params}`,
  {
    headers: {
      'X-API-Key': 'your-api-key-here'
    }
  }
);

const data = await response.json();
console.log(`Found ${data.total} profiles`);
console.log(data.profiles);
```

```python Python
import requests

headers = {
    'X-API-Key': 'your-api-key-here'
}

params = {
    'keyword': 'machine learning',
    'countryCode': 'US',
    'minFollowers': 500,
    'limit': 5,
}

response = requests.get(
    'https://api.fetchin.io/api/v1/profiles/search',
    headers=headers,
    params=params
)

data = response.json()
print(f"Found {data['total']} profiles")
print(data['profiles'])
```

```php PHP
<?php

$apiKey = 'your-api-key-here';

$params = http_build_query([
    'keyword' => 'machine learning',
    'countryCode' => 'US',
    'minFollowers' => 500,
    'limit' => 5,
]);

$url = "https://api.fetchin.io/api/v1/profiles/search?" . $params;

$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, $url);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'X-API-Key: ' . $apiKey
]);

$response = curl_exec($ch);
$data = json_decode($response, true);

curl_close($ch);
echo "Found " . $data['total'] . " profiles\n";
print_r($data['profiles']);
?>
```

## Notes

<Info>
  **Credits**: This endpoint charges `limit × 100` credits per request. For example, `limit=10` costs 1,000 credits. Only successful requests are charged.
</Info>

<Warning>
  Response times may vary depending on the complexity of your filters.
</Warning>

<Tip>
  Use `excludeUrns` to avoid duplicates when making multiple calls. Combine filters for precise targeting — for example, `jobTitle=CTO&companyName=&minFollowers=5000` finds high-profile CTOs.
</Tip>
